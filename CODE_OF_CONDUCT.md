# Code of Conduct

NodeCue is an early public alpha for a Blender add-on and related node skills. The project should stay useful for artists, technical directors, developers, and learners with different levels of Blender and agent-tool experience.

## Expected Behavior

- Be respectful and direct.
- Keep feedback specific and reproducible when possible.
- Assume alpha testers may be reporting confusing setup or model behavior, not making final product claims.
- Keep private API keys, asset-library paths, proprietary `.blend` files, and unreleasable screenshots out of public issues.
- Discuss paid setup or consulting privately after the public issue identifies the request.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory comments.
- Posting another person's private credentials, files, or asset-library details.
- Pressuring users to share proprietary `.blend` files publicly.
- Treating basic alpha access, bug reports, or fixes for broken documented behavior as paid-only work.

## Reporting

Use GitHub issues for ordinary project feedback. If a report involves private files, credentials, or sensitive commercial details, avoid posting those details publicly and ask for a private contact path instead.

Maintainers may edit, hide, or remove public content that exposes private credentials, private assets, or unrelated abusive material.
